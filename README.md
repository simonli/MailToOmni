# MailToOmni
Wox插件, C#组件，发送邮件到OmniFocus的MailDrop

> - 可配置发送邮件服务器，用户名，密码

> - 可配置OmniFocus的MailDrop地址

> - 可自定义任务及注释分隔符

# Usage
> 激活键t，然后输入： 任务标题 **&** 任务注释。 其中“**&**” 在配置文件配置
